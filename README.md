# Домашняя работа 14.2

### Реализация:
В проекте инициализировано 2 класса: `Product` и `Category`.
`Product`: Принимает список из имени, описания, цены и количества.
`Category`: Принимает список из имени категории, описания, списка продуктов.

### Новые функции проекта:
1. Возможность добавлять продукты в категории
2. Дубликаты продуктов автоматически добавляются к уже существующему объекту, цена при этом меняется на большую
3. Защита от нулевой и отрицательной цены
