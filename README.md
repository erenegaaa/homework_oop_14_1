# Домашняя работа 14.2

### Реализация:
В проекте инициализировано 2 класса: `Product` и `Category`.
`Product`: Принимает список из имени, описания, цены и количества.
`Category`: Принимает список из имени категории, описания, списка продуктов.
#### Функции в проекте:
Возможность добавлять продукты в категории
Дубликаты продуктов автоматически добавляются к уже существующему объекту, цена при этом меняется в большую сторону.
Защита от нулевой и отрицательной цены

### Нововведение:
1. Вывод на экран осуществляется при помощи магического метода `__str__`
Для категории: "Название продукта", "Количество" шт.
Для продукта: "Название продукта", "Описание" цена, количество
2. Добавлена возможность считать стоимость продуктов при помощи `__add__`
Пример вызова: print(product2 + product3)
